Holistic recommendations — meta-ensemble across methodologies

Each row is a (ticker, horizon) recommendation from the holistic meta-ensemble. The meta-ensemble runs every sub-methodology (trend-following, mean-reversion, etc.), drops any below 50% backtest accuracy, and stacks the survivors' votes weighted by their accuracy. Only emits a call when at least 2 methodologies agree and the net vote margin clears 15%. This is the recommendation to act on — sub-methodology breakdown is shown in the expand cell.

Methodologies — different ways to combine the same patterns

Each methodology is a different rule for combining patterns into a prediction. They're all evaluated against the same set of backtest samples, so any difference in accuracy is a difference in approach, not data. The "best" methodology is the one we should weight most when combining signals in production.

Backtest accuracy — random historical samples

The engine picks random (ticker, past-date) samples, runs the same pipeline using ONLY data available on that date, then compares to the actual N-day forward move. This is how we validate signal quality without waiting weeks. Headline accuracy is directional: it counts only samples where the ensemble made an up/down call. Neutrals are reported as neutral rate.

How it works — what historical windows we use

The system is not looking at a single day of data when it makes a prediction. Every pattern detector uses a different historical window, and the ensemble combines them so the final call is informed by short, medium, and long term context simultaneously.

Short-term context (5–14 days)

  • RSI(14) — momentum over last 2 trading weeks
  • 5-day SMA for multi-timeframe alignment
  • Candlestick patterns: engulfing, hammer, shooting star (1–5 day windows)
  • ATR(14) for stop-loss distance

Medium-term context (20–50 days)

  • 20-day SMA, 20-day Bollinger Bands (squeeze detection over 60-day window)
  • MACD(12/26/9) — exponential 12 and 26 day EMAs
  • 50-day SMA — primary swing trend
  • Volume z-score over 20 bars
  • Doji at support/resistance (30-bar look-back for S/R)

Long-term context (200+ days)

  • 200-day SMA — long-term trend
  • Golden cross / death cross (50d vs 200d crossover)
  • Multi-timeframe alignment: price > 5/20/50/200 SMA stack
  • Market regime: SPY vs its own 200-day SMA + 60-bar slope

Cross-cutting

  • Per-horizon weight tuning: each pattern's weight is tuned separately for 5d, 10d, 20d predictions
  • Market regime tagging: each backtest sample is tagged with bull/bear/choppy regime so we can see edge by regime
  • News sentiment (live only): VADER on recent headlines via Yahoo Finance
  • Backtest cutoff date sees only data up to that date — strict no-look-ahead
